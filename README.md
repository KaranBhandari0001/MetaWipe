# 🧹 MetaWipe – Remove AI Metadata from Images

MetaWipe is a simple and powerful browser extension that removes all metadata from images — including AI-generated data, EXIF information, and hidden tags — before you upload or share them online.

---

## 🤔 Why I Built This

I was creating AI-generated posters and uploading them on LinkedIn.

But LinkedIn was detecting that the images were generated using AI and automatically adding labels or flags related to AI content and copyright.

To solve this problem, I built **MetaWipe** — a tool that removes all hidden metadata from images and makes them clean before uploading.

---

## ⚡ What It Does

MetaWipe removes:

- 📸 EXIF metadata (camera, location, time, etc.)
- 🤖 AI-generated metadata (prompt, seed, model info)
- 🧬 Hidden PNG text chunks (tEXt, iTXt, zTXt)
- 🛠 Editing software data (Photoshop, Canva, etc.)

---

## 🔥 Key Features

- ✅ One-click metadata removal  
- ✅ Works completely offline  
- ✅ Fast and lightweight  
- ✅ No data upload (privacy safe)  
- ✅ Clean and modern UI  

---

## 🛠 How It Works

MetaWipe uses a simple but powerful technique:

1. Upload an image  
2. The image is redrawn onto a canvas  
3. A new image is generated without any metadata  

👉 This process strips all hidden data automatically.

---

## 📦 Installation

1. Download or clone this repository  
2. Open your browser and go to:
3. Enable **Developer Mode**  
4. Click on **Load unpacked**  
5. Select the MetaWipe folder  

---

## 🖼 Usage

1. Open the extension  
2. Upload your image  
3. Click **Remove Metadata**  
4. Download the cleaned image  

---

## ⚠️ Limitations

MetaWipe removes metadata, but it does NOT:

- ❌ Remove visible watermarks  
- ❌ Remove pixel-level AI watermarks  
- ❌ Decode steganography  

---

## 🔐 Privacy

Your images are never uploaded anywhere.

Everything runs locally in your browser.
